Password Strength Checker Proof-of-Concept Lab

Readme

This lab demonstrates a simple password strength checker written in Python. It evaluates passwords based on length, lowercase letters, uppercase letters, digits, and special characters, providing a score and suggestions for improvement.

Password Strength Checker Script (password_strength_checker.py)

import string

def check_password_strength(password):
    """
    Simple password strength checker.
    Returns a score (0-5) and a description.
    """
    score = 0
    reasons = []

    # Length
    if len(password) >= 8:
        score += 1
    else:
        reasons.append("Password is too short (min 8 characters)")

    # Lowercase
    if any(c in string.ascii_lowercase for c in password):
        score += 1
    else:
        reasons.append("Add at least one lowercase letter")

    # Uppercase
    if any(c in string.ascii_uppercase for c in password):
        score += 1
    else:
        reasons.append("Add at least one uppercase letter")

    # Digit
    if any(c in string.digits for c in password):
        score += 1
    else:
        reasons.append("Add at least one digit")

    # Special character
    if any(c in string.punctuation for c in password):
        score += 1
    else:
        reasons.append("Add at least one special character (!@#$...)")

    # Strength description
    if score <= 2:
        desc = "Weak"
    elif score == 3:
        desc = "Moderate"
    elif score == 4:
        desc = "Strong"
    else:
        desc = "Very Strong"

    return score, desc, reasons

if __name__ == "__main__":
    test_password = input("Enter password to check: ")
    score, desc, reasons = check_password_strength(test_password)
    print(f"Password Score: {score}/5")
    print(f"Strength: {desc}")
    if reasons:
        print("Suggestions to improve:")
        for r in reasons:
            print("-", r)

Explanation

The script checks passwords against five criteria: length, lowercase letters, uppercase letters, digits, and special characters. It calculates a score from 0 to 5 and provides human-readable suggestions to improve weak passwords.
