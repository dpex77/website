README:

Log File Analyzer scans logs for **sensitive information leaks** such as plaintext passwords, API keys, authentication tokens, or secrets. This helps in detecting **data exposure vulnerabilities** in application/server logs.

Code:

import re

# User input
log_file = input("Enter log file path: ")

# Patterns
patterns = [
    r"password\s*=\s*.+",
    r"passwd\s*=\s*.+",
    r"secret\s*=\s*.+",
    r"apikey\s*=\s*.+",
    r"token\s*=\s*.+",
    r"AKIA[0-9A-Z]{16}",        # AWS Access Key ID
    r"eyJ[a-zA-Z0-9_-]{10,}"    # JWT token
]

print(f"Scanning log file {log_file}...\n")

with open(log_file, "r", errors="ignore") as f:
    for num, line in enumerate(f, 1):
        for pattern in patterns:
            if re.search(pattern, line, re.IGNORECASE):
                print(f"[Line {num}] {line.strip()}")

print("\nAnalysis complete.")
