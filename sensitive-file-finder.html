README:

Sensitive File Finder scans a directory recursively for files that are likely to contain sensitive information (e.g., config.php, .env, id_rsa, .bak backups). This is useful for identifying **credentials, keys, or misconfigurations** accidentally left on a server or source code repository.

Code:

import os

# User input
root_dir = input("Enter directory to scan: ")

# Sensitive filenames/extensions
sensitive_files = [
    "config.php", ".env", "settings.py", "db.conf",
    "id_rsa", "id_dsa", ".htpasswd", "web.config",
    "passwd", "shadow"
]
sensitive_exts = [".bak", ".old", ".zip", ".tar", ".gz"]

print(f"Scanning directory {root_dir}...\n")

for root, dirs, files in os.walk(root_dir):
    for file in files:
        if file in sensitive_files or any(file.endswith(ext) for ext in sensitive_exts):
            print(f"[!] Found sensitive file: {os.path.join(root, file)}")

print("\nScan complete.")
