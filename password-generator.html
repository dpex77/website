Password Generator Proof-of-Concept Lab

Readme

This lab demonstrates a secure password generator written in Python. It uses secrets instead of random for cryptographic security, ensures at least one character from each category (lowercase, uppercase, digit, special), and produces strong unpredictable passwords.

Password Generator Script (password_gen.py)

import secrets
import string
import random

def generate_password(length=8):
    # Password must be at least 4 characters
    if length < 4:
        raise ValueError("Password length must be at least 4")

    # Step 1: character sets
    lower = string.ascii_lowercase
    upper = string.ascii_uppercase
    digits = string.digits
    symbols = string.punctuation

    all_chars = lower + upper + digits + symbols

    # Step 2: ensure one of each
    lower_char = secrets.choice(lower)
    upper_char = secrets.choice(upper)
    digit_char = secrets.choice(digits)
    special_char = secrets.choice(symbols)

    password_chars = [lower_char, upper_char, digit_char, special_char]

    # Step 3: fill the rest
    remaining = length - 4
    for i in range(remaining):
        ch = secrets.choice(all_chars)
        password_chars.append(ch)

    # Step 4: shuffle
    random.shuffle(password_chars)

    # Step 5: join
    final_password = ''.join(password_chars)
    return final_password

if __name__ == "__main__":
    print("Generated password:", generate_password(12))

Explanation

The script guarantees that every generated password includes at least one lowercase letter, one uppercase letter, one digit, and one special character. By using secrets, the randomness is cryptographically strong, making the passwords suitable for security-sensitive use cases.
